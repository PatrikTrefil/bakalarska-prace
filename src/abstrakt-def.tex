\def\Abstrakt{%
    Tato bakalářská práce se zabývá vývojem webové aplikace pro monitorování mentálního zdraví.
    Aplikace byla vyvinuta ve spolupráci s Národním ústavem duševního zdraví.
    Platforma poskytuje terapeutům nástroje pro spolupráci s pacienty a klienty mimo běžné terapeutické sezení.
    Práce se skládá z analýzy požadavků, návrhu řešení, technické a uživatelské dokumentace.
    V rámci této práce byla implementována spolupráce formou dotazníků.
    Tato funkcionalita lze použít i pro sběr dat pro výzkumné účely.
    Aplikace byla vhodně navržena pro jednoduchou rozšiřitelnost o další formy spolupráce.
    Aplikace byla úspěšně dokončena a předána zadavateli.
}

\def\AbstraktEN{%
    This bachelor's thesis deals with the development of a web application for monitoring mental health.
    The application was developed in collaboration with the National Institute of Mental Health.
    The platform provides therapists with tools for collaborating with patients and clients outside of regular therapeutic sessions.
    The work consists of requirement analysis, solution design, technical and user documentation.
    Within this work, collaboration in the form of questionnaires was implemented.
    This functionality can also be used for data collection for research purposes.
    The application was appropriately designed for easy extension with additional forms of collaboration.
    The application was successfully completed and delivered to the client.
}
