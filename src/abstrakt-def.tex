\def\Abstrakt{%
    Tato bakalářská práce se zabývá vývojem webové aplikace pro Národní ústav duševního zdraví.
    Vytvořená platforma poskytuje terapeutům nástroje pro spolupráci s pacienty a klienty mimo běžné terapeutické sezení.
    V rámci této práce byla implementována spolupráce formou dotazníků.
    Tato funkcionalita lze použít i pro sběr dat pro výzkumné účely.
    Aplikace byla vhodně navržena pro jednoduchou rozšiřitelnost o další formy spolupráce.
    Aplikace je dobře připravena pro zkvalitnění poskytované péče a byla úspěšně předána zadavateli.
}

\def\AbstraktEN{%
    This bachelor's thesis deals with the development of a web application for the National Institute of Mental Health.
    The platform provides therapists with tools for collaborating with patients and clients outside of regular therapeutic sessions.
    The work consists of requirement analysis, solution design, technical and user documentation.
    Within this work, collaboration in the form of questionnaires was implemented.
    This functionality can also be used for data collection for research purposes.
    The application was designed for easy extension with additional forms of collaboration.
    The application is well prepared for the improvement of the provided care and was successfully delivered to the client.
}
